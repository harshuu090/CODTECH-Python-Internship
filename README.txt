Project Title: Weather API Integration and Data Visualization

Description:
This project integrates OpenWeatherMap API using Python and fetches real-time weather forecast data. The data is visualized using Matplotlib in the form of temperature and humidity graphs.

Technologies Used:
- Python
- Requests Library
- Matplotlib
- OpenWeatherMap API

Features:
- Fetches 24-hour forecast data
- Displays temperature trend
- Displays humidity levels
- Error handling for API connection

Outcome:
Successfully integrated public API and created visualization dashboard as required in internship task.
